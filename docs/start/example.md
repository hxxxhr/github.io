<h3> 功能示例 使用介绍 </h3>

## 说明

功能示例项目，是基于[Mars2D 平台](http://mars2d.cn/)做的一个按每个单独功能进行单页面展示、并且支持修改编辑代码实时运行的一个演示功能的系统。 主要用于开发人员学习了解 Mars2D 每个功能的使用、用于业务人员了解体验 Mars2D 具备的功能点。

## 选择技术栈

请根据您的使用技术栈，选择其中一种进行下载学习即可。
