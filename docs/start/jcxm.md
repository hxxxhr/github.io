<h3> 基础项目 使用介绍 </h3>

## 说明

基础项目 是基于[Mars2D 平台](http://mars2d.cn/)做的一个应用系统，提供的一个样板房模版。 包含常用基础地图功能，可在该基础项目上快速开发搭建新项目。方便快速搭建三维地图产品，敏捷开发，可复用，支持各种配置，适合各种场景使用。
